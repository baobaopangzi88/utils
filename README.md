### /linux  
* **rm_redirect** 修改rm命令  
   因为rm和mv很相似,修改后可以避免很多麻烦。如果确实要删除，可以用/bin/rm  
* **virtual_memory** 增加虚拟内存  
   多用于极耗内存的一次性的运算。
